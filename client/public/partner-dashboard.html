🔌 API Integration Quick Start

Use your partner API key to programmatically register clients, place orders, and check positions. All endpoints use the X-API-Key header.

1. Register a Client Under Your Partner
POST /api/v1/clients Headers: X-API-Key: T1P_your_partner_key Body: { "firstName": "John", "lastName": "Doe", "email": "john@example.com", "country": "US", "clientType": "retail" }
2. Place an Order for a Client
POST /api/v1/orders Headers: X-API-Key: T1P_your_partner_key Body: { "clientId": "uuid-from-step-1", "instrumentId": 1, "side": "buy", "orderType": "market", "quantity": 100 }
3. Check Client Positions
GET /api/v1/positions?clientId=uuid-from-step-1 Headers: X-API-Key: T1P_your_partner_key
4. Webhook Events

Configure your webhook URL in API Config to receive real-time events: order.filled, order.rejected, kyc.approved, kyc.rejected, transfer.completed, client.created. Each event includes a partnerId and timestamp.

5. Custom Domain Setup

Point your CNAME record (e.g. trade.yourdomain.com) to t1broker.com. The platform auto-detects your domain and serves your branded UI with your logo, colors, and support links.
